Lecture 02: Operators, Conditional Statements & Switch

Is lecture mein hum C++ ke sabhi operators ko detail mein cover karenge — arithmetic se lekar bitwise tak. Saath mein if-else conditional statements, nested conditions, aur switch statement ka complete understanding develop karenge. Ye concepts C++ programming ke building blocks hain.

Operators

Operators special symbols hote hain jo variables aur values par operations perform karte hain. C++ mein operators ko categories mein divide kiya jata hai based on unka kaam.

Arithmetic Operators

Ye operators mathematical calculations ke liye use hote hain. Inme basic operations like addition, subtraction, multiplication, division, aur modulo (remainder) shamil hain.

int a = 10, b = 3; cout << a + b; // 13 (addition) cout << a - b; // 7 (subtraction) cout << a * b; // 30 (multiplication) cout << a / b; // 3 (division - integer division) cout << a % b; // 1 (remainder/modulo)
Edge Case Integer Division Trap: Jab do integers ko divide karte hain, to result bhi integer hota hai. int a = 7, b = 2; cout << a / b; // Output: 3 (not 3.5) // For decimal result, use double: double result = (double)a / b;
int marks1 = 80, marks2 = 90; int total = marks1 + marks2; int average = total / 2; cout << "Total: " << total << endl; // 170 cout << "Average: " << average << endl; // 85

Assignment Operators

Assignment operators values assign karne ke liye use hote hain. Compound assignment operators (+=, -=, etc.) code ko concise banate hain.

int x = 10; x = 5; // x becomes 5 x += 3; // x = x + 3 → x becomes 8 x -= 2; // x = x - 2 → x becomes 6 x *= 4; // x = x * 4 → x becomes 24 x /= 2; // x = x / 2 → x becomes 12 x %= 5; // x = x % 5 → x becomes 2

Comparison Operators

Comparison operators do values ko compare karte hain aur result true ya false (boolean) return karte hain. Ye conditional statements mein bahut use hote hain.

OperatorNameExampleResult

!=Not equal to10 != 5true
>Greater than10 > 5true

Important == aur = mein confusion common hai. == comparison hai (equals?), jabki = assignment hai (store value). if (x = 5) // WRONG: assigns 5 to x, always true if (x == 5) // CORRECT: checks if x equals 5

Logical Operators

Logical operators multiple conditions ko combine karne ke liye use hote hain. Ye boolean expressions par kaam karte hain.

OperatorNameRuleExample
&&ANDBoth true → truetrue && false = false
||ORAt least one true → truetrue || false = true
!NOTReverse the value!true = false
int age = 20; bool hasLicense = true; // Can drive if age >= 18 AND has license if(age >= 18 && hasLicense) { cout << "Can drive!"; } // Short-circuit evaluation: && mein agar first false, second check nahi hota // || mein agar first true, second check nahi hota

Increment and Decrement Operators

Ye operators variable ki value ko 1 se badhane ya ghataane ke liye use hote hain. Inme prefix aur postfix do forms hoti hain.

int x = 5; x++; // Post-increment: x becomes 6 (pehle use, phir badhao) ++x; // Pre-increment: x becomes 7 (pehle badhao, phir use) x--; // Post-decrement: x becomes 6 --x; // Pre-decrement: x becomes 5
Edge Case Prefix vs Postfix Difference: int a = 5, b; b = a++; // b = 5, a = 6 (pehle b mein a ki old value, phir a++ ) int c = 5, d; d = ++c; // d = 6, c = 6 (pehle c++ , phir d mein new value) // Same concept -- ke liye bhi apply hota hai

Bitwise Operators

Bitwise operators numbers ke individual bits par directly operate karte hain. Ye low-level programming, flags, permissions, aur performance-critical code mein bahut use hote hain.

Bitwise AND, OR, XOR, NOT

OperatorNameRuleExample (5 & 3)
&Bitwise ANDBoth 1 → 1, else 05 (101) & 3 (011) = 1 (001)
|Bitwise ORAt least one 1 → 15 (101) | 3 (011) = 7 (111)
^Bitwise XORDifferent → 1, Same → 05 (101) ^ 3 (011) = 6 (110)
~Bitwise NOT1 → 0, 0 → 1~5 (00000101) = -6 (11111010)
int a = 5; // Binary: 00000101 int b = 3; // Binary: 00000011 cout << (a & b) << endl; // 1 (00000001) cout << (a | b) << endl; // 7 (00000111) cout << (a ^ b) << endl; // 6 (00000110) cout << (~a) << endl; // -6 (Two's complement)

Bitwise Shift Operators

Shift operators bits ko left ya right move karte hain. Ye multiplication aur division by powers of 2 ke equivalent hote hain.

OperatorNameEffectExample
<<Left ShiftBits left move, zeros fill right5 << 1 = 10 (101 → 1010)
>>Right ShiftBits right move, zeros fill left5 >> 1 = 2 (101 → 010)
int x = 5; // Binary: 00000101 cout << (x << 1) << endl; // 10 (00001010) → 5 * 2^1 cout << (x << 2) << endl; // 20 (00010100) → 5 * 2^2 cout << (x >> 1) << endl; // 2 (00000010) → 5 / 2^1 cout << (x >> 2) << endl; // 1 (00000001) → 5 / 2^2
Key Insight: Left shift n positions = multiply by 2^n. Right shift n positions = divide by 2^n. Ye operators hardware-level multiplication se fast hote hain, isliye embedded systems aur graphics programming mein widely use hote hain.
Intuition Bitwise AND & ko socho mask ki tarah. Jahan mask mein 1 hai, wahan original bit pass hoti hai. Jahan 0 hai, wahan 0 aa jata hai. Isliye permissions check karne ke liye AND use hota hai.

Bitwise OR | ko socho combine karne ki tarah. Do bit patterns ko merge karo — jahan kisi mein bhi 1 hai, result mein 1 aa jayega. Isliye permissions set karne ke liye OR use hota hai.
Interview Tip Bitwise operators se frequently puche jate hain:
1. Swap two numbers without temp variable: a = a ^ b; b = a ^ b; a = a ^ b;
2. Check if number is power of 2: (n & (n-1)) == 0
3. Check odd/even: (n & 1) == 0 → even, else odd
4. Turn off k-th bit: n & ~(1 << k)
5. Turn on k-th bit: n | (1 << k)
6. Toggle k-th bit: n ^ (1 << k)

Input and Output

Output with cout

cout (character output) screen par data print karne ke liye use hota hai. endl new line insert karta hai aur buffer flush karta hai. << insertion operator hai.

cout << "Hello World"; cout << "Age: " << 25; cout << "Name: " << name << endl; // endl = new line + flush

Input with cin

cin (character input) user se keyboard input lene ke liye use hota hai. >> extraction operator hai. Ye whitespace (space, tab, newline) ko delimiter manta hai.

int age; cout << "Enter your age: "; cin >> age; string name; cout << "Enter your name: "; cin >> name; // Only reads one word!

Complete Input/Output Example

#include <iostream> using namespace std; int main() { string name; int age; cout << "Enter your name: "; cin >> name; cout << "Enter your age: "; cin >> age; cout << " Hello " << name << "!" << endl; cout << "You are " << age << " years old." << endl; return 0; }

Conditional Statements

Conditional statements program ko decisions lene ki ability dete hain. Code ka flow conditions ke basis par change hota hai.

Basic If Statement

Simplest form — agar condition true hai to code block execute hoga, warna nahi.

if(condition) { // code runs if condition is true }
int age = 20; if(age >= 18) { cout << "You are an adult"; }

If-Else Statement

Do paths provide karta hai — condition true ho to if block, false ho to else block.

if(condition) { // code runs if condition is true } else { // code runs if condition is false }
int age = 15; if(age >= 18) { cout << "You are an adult"; } else { cout << "You are a minor"; }

If-Else If-Else Ladder

Jab multiple conditions check karni ho, tab if-else if ladder use hoti hai. Conditions top se bottom check hoti hain, aur pehli true condition ka block execute hota hai.

if(condition1) { // runs if condition1 is true } else if(condition2) { // runs if condition2 is true } else { // runs if all conditions are false }
int marks = 85; if(marks >= 90) { cout << "Grade: A"; } else if(marks >= 80) { cout << "Grade: B"; } else if(marks >= 70) { cout << "Grade: C"; } else if(marks >= 60) { cout << "Grade: D"; } else { cout << "Grade: F"; }
Best Practice Conditions hamesha most specific se least specific ki taraf rakho. Agar marks >= 60 pehle check kiya aur marks 85 hain, to wo Grade D de dega, jo galat hai. Isliye descending order (highest threshold first) follow karo.

Nested If Statements

Ek if statement ke andar doosra if statement — jab multiple conditions dependent hain.

int age = 20; bool hasLicense = true; if(age >= 18) { if(hasLicense) { cout << "You can drive"; } else { cout << "You need a license"; } } else { cout << "You are too young"; }
Key Insight: Nested if ko combine karke ek single if mein likh sakte hain: if(age >= 18 && hasLicense). Ye code readable aur maintainable banata hai. Nested if sirf tab use karo jab intermediate actions bhi chahiye ho.

Switch Statement

What is Switch

Switch statement multiple if-else if ladder ka alternative hai. Ye ek variable ko multiple constant values se compare karta hai. Switch generally tab use hota hai jab ek hi variable ko different fixed values se check karna ho.

switch(variable) { case value1: // code break; case value2: // code break; default: // code if no match }

Example: Days of Week

int day = 3; switch(day) { case 1: cout << "Monday" << endl; break; case 2: cout << "Tuesday" << endl; break; case 3: cout << "Wednesday" << endl; break; case 4: cout << "Thursday" << endl; break; case 5: cout << "Friday" << endl; break; case 6: cout << "Saturday" << endl; break; case 7: cout << "Sunday" << endl; break; default: cout << "Invalid day" << endl; }

Example: Calculator with Switch

char op; double a, b; cout << "Enter operator (+, -, *, /): "; cin >> op; cout << "Enter two numbers: "; cin >> a >> b; switch(op) { case '+': cout << "Result: " << (a + b) << endl; break; case '-': cout << "Result: " << (a - b) << endl; break; case '*': cout << "Result: " << (a * b) << endl; break; case '/': if(b != 0) { cout << "Result: " << (a / b) << endl; } else { cout << "Cannot divide by zero!" << endl; } break; default: cout << "Invalid operator" << endl; }

Fall-Through Behavior

Switch mein break statement bahut important hai. Agar break nahi likha, to control next case mein bhi chala jata hai — isse fall-through kehte hain.

int num = 2; switch(num) { case 1: cout << "One" << endl; case 2: cout << "Two" << endl; // Prints this case 3: cout << "Three" << endl; // Also prints this! default: cout << "Other" << endl; // And this too! } // Output: Two // Three // Other
Intuition Fall-through intentionally bhi use hota hai! Jaise multiple cases mein same code execute karna ho: case 1: case 2: case 3: cout << "Weekday"; break; case 6: case 7: cout << "Weekend"; break;
Important Switch mein sirf integer, char, aur enum types use kar sakte hain. Float, double, ya string direct use nahi kar sakte. Isliye switch ka use limited hota hai compared to if-else. C++ mein switch ke cases compile-time constants hone chahiye.

Key Takeaways

Takeaway
  • Arithmetic operators basic math karte hain; integer division aur modulo ke behavior ko carefully samajhna zaroori hai
  • Comparison operators true/false return karte hain; == aur = mein hamesha difference rakho
  • Logical operators &&, ||, ! multiple conditions combine karne mein use hote hain; short-circuit evaluation performance improve karti hai
  • Prefix ++x pehle increment karta hai, postfix x++ baad mein — assignment ke time difference matter karta hai
  • Bitwise operators bits par directly operate karte hain; left shift << multiply by 2^n, right shift >> divide by 2^n
  • cin whitespace ko delimiter manta hai; spaces ke saath input ke liye getline() use karo
  • If-else ladder mein conditions most specific se least specific rakho
  • Switch sirf equality check ke liye use hota hai; break statement fall-through rokta hai
  • Nested if ko logical operators se combine karke readable code likh sakte hain

Interview Tip Commonly asked interview questions:
1. == aur = mein difference: == comparison, = assignment. if(x = 5) compile hoga lekin logical bug hoga.
2. Short-circuit evaluation: && mein agar first false, second evaluate nahi hota. || mein agar first true, second evaluate nahi hota. Iska use null pointer checks mein hota hai.
3. Switch vs If-else: Switch jump table use karta hai, isliye multiple cases mein faster. If-else flexible hai koi bhi condition check kar sakta hai.
4. Bitwise use cases: Permission flags, graphics (RGB values), compression algorithms, embedded systems.
5. Pre vs Post increment: Prefix pehle increment karta hai (use karne se pehle), postfix baad mein (use karne ke baad).
