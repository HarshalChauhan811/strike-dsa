Lecture 07: Introduction to Arrays

Is lecture mein hum Arrays ko bilkul shuru se samajhenge — kya hai, kyun zaroori hai, memory mein kaise store hote hain, aur kaise use karte hain. Array programming ka sabse important data structure hai. Agar aap first time arrays padh rahe ho, toh ye notes aapke liye perfect hai.

❦ ❦ ❦

The Problem — Why We Need Arrays

Socho aapko 5 students ke marks store karne hain. Bina array ke aapko 5 alag-alag variables banane padenge:

int marks1 = 70; int marks2 = 40; int marks3 = 80; int marks4 = 14; int marks5 = 18; // Print karna ho toh 5 baar cout cout << marks1 << endl; cout << marks2 << endl; // ... aur aage 3 aur lines
Important Ab socho 100 students ke marks store karne hain! Bina array ke 100 variables banane padenge — marks1, marks2, marks3... marks100. Ye impossible hai. Isliye Array aata hai bachane.
Key Insight: Array ek aisa container hai jisme ek hi type ke multiple values ek saath store ho sakti hain — aur unhe ek index number se access kiya ja sakta hai.
❦ ❦ ❦

What is an Array?

Array ek collection of similar data types hai jo continuous memory locations mein store hote hain. Har element ko ek index number se access kiya ja sakta hai.

Intuition Array ko ek building ki floors ki tarah samjho. Har floor pe ek family rehti hai (value). Ground floor = 0, First floor = 1, Second floor = 2... Har floor ka ek unique number hai jisse aap directly us floor pe pahunch sakte ho. Array mein ye number index hota hai.

Array Declaration

int marks[5]; // 5 integers ka array declare kiya

Is line ka matlab:

  • int → Array mein integers store honge
  • marks → Array ka naam
  • [5] → Array mein 5 elements aa sakte hain

Array Initialization

Method 1: Index by Index

int marks[5]; // Array declare marks[0] = 70; // Index 0 pe 70 daala marks[1] = 40; // Index 1 pe 40 daala marks[2] = 80; // Index 2 pe 80 daala marks[3] = 14; // Index 3 pe 14 daala marks[4] = 18; // Index 4 pe 18 daala

Method 2: Direct Initialization

int marks[5] = {70, 40, 80, 14, 18}; // Ya phir size bhi chhod sakte ho int marks[] = {70, 40, 80, 14, 18}; // Compiler khud size 5 le lega
Workflow
  • Step 1: Array declare karo — int marks[5];
  • Step 2: Values daalo — marks[0] = 70;
  • Step 3: Values access karo — cout << marks[0];

0-Based Indexing — Very Important

Important C++ mein array hamesha 0 se start hota hai. Agar aapne int arr[5] declare kiya, toh valid indexes hain: 0, 1, 2, 3, 4. arr[5] access karna galat hai — ye array ke bahar hai.

Key Insight: Agar array ka size n hai, toh last valid index n-1 hota hai. int arr[5] ke liye last index 4 hai, 5 nahi. Ye sabse common beginner mistake hai.

Traversing an Array (Loop se)

Array ke sabhi elements ko ek-ek karke access karna traversing kehlata hai. Iske liye for loop sabse best hai.

#include <iostream> using namespace std; int main() { int marks[5] = {70, 40, 80, 14, 18}; // Array ko traverse kar rahe hain for(int i = 0; i < 5; i++) { cout << "marks[" << i << "] = " << marks[i] << endl; } return 0; }

Array Input and Output

User se Input Lena

#include <iostream> using namespace std; int main() { int n; cout << "Tell me the size of an Array: "; cin >> n; int arr[n]; // User ke bataye size ka array cout << "Enter the values in an Array:" << endl; // Input loop for(int i = 0; i < n; i++) { cin >> arr[i]; } // Output loop cout << "Array elements are:" << endl; for(int i = 0; i < n; i++) { cout << arr[i] << " "; } return 0; }
Edge Case Warning: Kuch compilers mein int arr[n]; (variable size) allow nahi hota. Isliye constant size use karo — int arr[100]; aur phir actual n elements hi input lo. Ya phir dynamic allocation use karo (baad mein padhenge).

Memory Layout — How Array is Stored

Array continuous memory locations mein store hota hai. Har element ka address pichle element ke address se data type ke size jitna door hota hai.

int Array (4 bytes each)

🔷 MEMORY LAYOUT: int arr[5] — Each int = 4 bytes

Index 0 1 2 3 4
Address 0x1000 0x1004 0x1008 0x100C 0x1010
Value 70 40 80 14 18
Size Total = 5 × 4 = 20 bytes

Address Calculation Formula

Intuition Kisi bhi element ka address nikalne ka formula:

Address of arr[i] = Base Address + (i × Size of Data Type)

Jaise arr[3] ka address: 1000 + (3 × 4) = 1000 + 12 = 1012 (0x100C)

double Array (8 bytes each)

🔷 MEMORY LAYOUT: double salary[5] — Each double = 8 bytes

Index 0 1 2 3 4
Address 0x2000 0x2008 0x2010 0x2018 0x2020
Value 50000.0 60000.0 45000.0 70000.0 55000.0
Size Total = 5 × 8 = 40 bytes

Har element 8 bytes door hai — kyunki double 8 bytes ka hota hai

Problem 1: Find Highest Element

Intuition Sabse pehle maan lo pehla element sabse bada hai. Phir baaki sab elements se compare karo. Agar koi element current highest se bada hai, toh usse new highest bana do.
#include <iostream> using namespace std; int main() { int marks[5] = {70, 80, 40, 90, 20}; // Step 1: Maan lo pehla element highest hai int highest = marks[0]; // Step 2: Baaki elements se compare karo for(int i = 1; i < 5; i++) { if(marks[i] > highest) { highest = marks[i]; // New highest mil gaya } } cout << "Highest marks: " << highest << endl; return 0; }

Problem 2: Search in an Array (Linear Search)

Intuition Array ke har element ko ek-ek karke check karo. Agar target element mil jaye, toh "Found" print karo aur loop se bahar niklo. Agar pura array check karne ke baad bhi na mile, toh "Not Found".
#include <iostream> using namespace std; int main() { int arr[5] = {70, 80, 40, 90, 20}; int search = 90; for(int i = 0; i < 5; i++) { if(arr[i] == search) { cout << "YES, Element found at index " << i << endl; return 0; // Program yahin khatam } } // Agar loop complete ho gaya aur element nahi mila cout << "NO, Element not found" << endl; return 0; }

Key Takeaways

Takeaway
  • Array Definition: Same type ke elements ka collection jo continuous memory mein store hote hain.
  • Declaration: int arr[5]; — 5 integers ka array.
  • 0-Based Indexing: Array hamesha 0 se start hota hai. Last index = size - 1.
  • Initialization: int arr[] = {1, 2, 3}; — compiler size khud le leta hai.
  • Memory Layout: Continuous addresses. Har element sizeof(data_type) bytes door.
  • Address Formula: Base Address + (index × size)
  • Traversal: for(int i = 0; i < n; i++) — sabse common pattern.
  • Input/Output: Loop se cin >> arr[i] aur cout << arr[i]
  • Highest Element: Pehle element ko highest maan lo, phir compare karte jao.
  • Linear Search: Har element check karo, milne pe return kar do.
